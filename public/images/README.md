# Image Placeholders

Your pipeline should provide images in the following locations:

- `/images/hero-main.png` - Homepage hero image
- `/images/problem-1.png`, `problem-2.png`, `problem-3.png` - Problem section images
- `/images/services/service-1.png` through `service-6.png` - Service card images
- `/images/blog/blog-1.png` through `blog-N.png` - Blog featured images

All image paths are referenced in the data files and can be customised per site.
